# cf_ai_studybuddy

An AI-powered spaced repetition quiz study app built on Cloudflare's developer platform. Upload a `.txt` study document and get automatically generated quiz questions using Llama 3.3 on Workers AI, then get your reviews scheduled using the SM-2 spaced repetition algorithm — the same algorithm behind Anki. Each answer is evaluated by the LLM, which provides a quality score and one-sentence feedback. Session state and study history are stored per-user in a Durable Object with SQLite, so your progress persists across sessions.

Deployed link: https://cf-ai-studybuddy.rubaosman10.workers.dev/

- A sample `.txt` file is included at `samples/ch3_study_guide.txt` for quick testing


## Features

- **Automatic question generation** — upload a `.txt` file and get 3–5 quiz questions per text chunk (roughly 15-20 questions total depending on material length), generated by Llama 3.3
- **AI answer evaluation** — answers are graded by the LLM on a 0–5 quality scale with contextual feedback
- **SM-2 spaced repetition** — review intervals adapt based on your performance, just like Anki
- **Session-complete flow** — once all due cards are reviewed, choose to stop or keep reviewing older cards
- **Progress page** — mastery percentage, due today/upcoming, difficulty breakdown, average memory strength
- **Per-user state** — each browser session gets its own Durable Object with isolated SQLite storage
- **Async document processing** — question generation runs as a background Workflow; the UI polls for readiness before redirecting

## Tech Stack

**Frontend**
- React + React Router v7 (SPA mode)
- Tailwind CSS
- Vite + Cloudflare Vite plugin

**Backend**
- Hono — REST API routing on Cloudflare Workers
- Cloudflare Workers AI — Llama 3.3 70B (`@cf/meta/llama-3.3-70b-instruct-fp8-fast`) for question generation and answer evaluation
- Cloudflare Durable Objects — per-session state with SQLite storage (questions, SM-2 cards, materials)
- Cloudflare Workflows — durable background job for document chunking and question generation

**Tooling**
- Wrangler CLI for deployment

**Request flow:**
```
Upload .txt → POST /api/upload → Workflow created (async)
                                      ↓
                          chunk → AI generate questions → store in DO
                                      ↓
Poll /api/status/:materialId → ready → redirect to /quiz
                                      ↓
GET /api/quiz/next → DO returns due card (SM-2 scheduled)
POST /api/quiz/answer → DO evaluates with AI → updates SM-2 card
```

## Setup & Running Locally

### Prerequisites
- Node.js 18+
- A Cloudflare account (free tier works)
- Wrangler CLI authenticated: `npx wrangler login`

### Install

```bash
git clone https://github.com/ruba0s/cf_ai_studybuddy.git
cd cf_ai_studybuddy
npm install
```

### Run locally

```bash
npm run dev
```

This builds the project and starts the local dev server with Wrangler, including local emulation of Workers AI, Durable Objects, and Workflows. Open [http://localhost:5173](http://localhost:5173).
